Al ejecutar el comando node server.js, Node.js inicia la aplicación y pone en funcionamiento el servidor.

En la terminal aparece el mensaje:
"Servidor ejecutándose en puerto 3000".

Cuando se visita la ruta /estado, el servidor responde con información en formato JSON indicando que el servicio se encuentra funcionando correctamente.

Respuesta obtenida:

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}
